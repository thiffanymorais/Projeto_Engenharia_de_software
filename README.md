# Novo Sistema de Pesquisa - Biblioteca FECAP

Este projeto visa a modernização do sistema de busca e gestão de empréstimos da biblioteca da Fundação Escola de Comércio Álvares Penteado (FECAP). O objetivo é elevar a satisfação do usuário através de uma interface intuitiva, responsiva e integrada à grade acadêmica.

## 🚀 Sobre o Projeto
O sistema atual apresenta limitações significativas em usabilidade e precisão nos resultados. Nosso projeto propõe uma arquitetura escalável e uma experiência focada no usuário (UX/UI).



## 🛠 Tecnologias Utilizadas
* **Back-end:** C# com .NET (ASP.NET Core)
* **Arquitetura:** Clean Architecture
* **Front-end:** [Inserir Framework, ex: Blazor/React]
* **Banco de Dados:** SQL Server
* **Análise de Dados:** Power BI / Looker Studio
* **Gestão de Projeto:** GitHub Projects (Kanban)

## 📋 Funcionalidades Principais
- [ ] **Busca Inteligente:** Autocompletar e filtros avançados.
- [ ] **Disponibilidade Real-time:** Status dos exemplares em tempo real.
- [ ] **Dashboard Acadêmico:** Listagem automática de livros obrigatórios.
- [ ] **Reserva/Renovação Online:** Self-service para empréstimos.
